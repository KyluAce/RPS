$$ X \sim N ( m, \sigma )$$
$$ m = 20, \ \ \sigma = 2 $$
$$ X \sim N ( 20, 2 )$$
$$ Z = \frac{X - 20}{2} $$  \\

$a)$ 
$$ P( X \ge 25,66 ) = 1 - P( X < 25,66 ) = $$
$$ = 1 - P(z < 2,83) = 1 - \Phi(2,83) = $$
$$ = 1 -  0,99767 = 0.00233$$ \\

$b)$ 
$$ P( X < 16,48 ) =  P( Z < -1,76 ) = $$
$$ = \Phi(-1,76) = 1 - \Phi(1,76) = $$
$$ = 1 -  0,86080 = 0.0392$$ \\

$c)$ 
$$ P( X \le 0 ) =  P( Z \le \frac{-20}{2} ) = $$
$$ = P( Z \le -10) = 1 - \Phi(10) = 0 $$
